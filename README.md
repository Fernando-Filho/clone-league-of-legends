## 🖱 Funcionalidades do jogo adotadas no projeto

-   ***Tela de login***: Redireciona o usuário para a página inicial do cliente após o login.
-   ***Social***: A funcionalidade de abrir e fechar pastas está implementada, mas não é possível abrir chats de conversa ou interagir diretamente.
-   ***Client***
    -   ***Outro céu***: Uma página dedicada aos desenvolvedores.
-   Loja
    -   ***Destaques***: Fornece informações sobre produtos com desconto ou recursos especiais na loja. 
    -   ***Campeões***: Permite filtrar campeões pelo nome, pela função (role) e verificar se estão em promoção. 


## ⚙ Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com) e um editor de texto como o [VSCode](https://code.visualstudio.com/), com a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).


## 🚀 Como executar o Projeto

O projeto é composto por duas partes principais: o Front-End e o Back-End com a [API](https://github.com/ViniPorto/lolchamps) para consultar os campeões.

#### 💻 Rodando o Front-End

1. Clone este repositório
```bash
$ git clone https://github.com/mirellanaspolini/clone-league-of-legends.git
```

2. Acesse a pasta do projeto no terminal/cmd
```bash
$ cd clone-league-of-legends
```

3. Abra a pasta do projeto no vscode
```bash
$ code .
```

4. Abra o arquivo index.html da pasta login, clique com o botão direito do mouse no editor e clique em Open with Live Server, ou pressione o atalho Alt+L alt+O

5. O servidor iniciará na porta:5500 - acesse http://localhost:5500/login

6. Use as seguintes credenciais para fazer login:

    * Usuário: user10
    * Senha: senhaforte55

#### 🎲 Rodando a API

Para saber como rodar a API acesse [este link](https://github.com/ViniPorto/lolchamps/blob/master/README.md#-como-executar-o-projeto).
